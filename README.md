# Infosec Portfolio

Simple two-page static portfolio for GitHub Pages:
- `index.html` - bio, skills, highlights, contact.
- `projects.html` - case studies and project details.
- `styles.css` - dark InfoSec style.

## Deploy to GitHub Pages

1. Create a new GitHub repository.
2. Commit and push these files.
3. In repo settings, enable GitHub Pages from main branch `/ (root)`.
4. Visit the published URL.

## Customize

- Replace placeholder name/email/links.
- Add real project details and skill evidence (reports, CTF write-ups, code links).
- Keep it static (good for secure airflow and infosec portfolio hygiene).
